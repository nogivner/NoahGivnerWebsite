MonkeyWrench Game Engine

A miniature 2D game engine built with C++. Features include Actor-Component system, physics with Rigidbody, audio management, particle systems, and comprehensive scene management.

VIEW ON GITHUB

FEATURES

Engine Capabilities

MonkeyWrench is a powerful 2D game engine designed for flexibility and performance.

Actor-Component System

Flexible and modular architecture with entity-component design pattern for easy game object composition.

Physics System

Rigidbody components, collider system, and raycast functionality powered by Box2D for realistic physics simulation.

Audio Management

Comprehensive audio system with AudioDB for sound management and playback control.

Particle Systems

Robust particle system for visual effects, explosions, and environmental elements.

Asset Management

ImageDB for efficient asset loading and management with support for various image formats.

Lua Scripting

luaBridge integration for runtime scripting capabilities and gameplay logic.

Event Bus

EventBus system for decoupled communication between game systems and components.

2D Rendering

Efficient renderer for 2D graphics with support for sprites, textures, and layered rendering.
